# Veil Messenger

**Говори свободно.**

Veil — это P2P-мессенджер с абсолютной приватностью. Zero Knowledge. Zero Trust.

## Возможности

- 🔐 **XChaCha20-Poly1305** — военное шифрование каждого сообщения
- 🔑 **Ed25519 ключи** — твоя личность это математика
- 📱 **P2P** — нет серверов, нет логов, нет метаданных
- 👥 **Группы** — защищённые групповые чаты
- 🎯 **Кодовые слова** — проверка безопасности канала
- 🖼️ **Фото-аватар** — загрузи свою картинку
- 😀 **Смайлики** — панель эмодзи
- 💬 **Ответы, удаление, закрепление** — всё для удобства
- 🌙 **Тёмная тема Obsidian** — стильный чёрный UI
- 🛡️ **2FA** — TOTP через Google Authenticator
- 🔄 **Восстановление** — seed-фраза из 24 слов

## Технологии

| Слой | Стек |
|------|------|
| Фреймворк | Flutter |
| Криптография | XChaCha20-Poly1305, Ed25519, BIP39, TOTP |
| Хранение | Hive (локальная БД) |
| UI | Material 3, Google Fonts |
